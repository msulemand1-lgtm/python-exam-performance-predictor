# Python Exam Performance Predictor

A logistic regression model that predicts whether a student will pass a Python exam based on their study habits and confidence levels.

## Project Structure
- `data-processing.py` — Cleans the raw dataset
- `model_training.py` — Trains the logistic regression model
- `prediction.py` — Interactive script to predict a single student's result
- `exploration_and_analysis.ipynb` — EDA and visualizations 

## How to Run
1. Install dependencies: pip install -r requirements.txt
2. Run data processing: python data-processing.py
3. Train the model: python model_training.py
4. Make a prediction: python prediction.py

## Model Performance
 Accuracy: ~78.6%
